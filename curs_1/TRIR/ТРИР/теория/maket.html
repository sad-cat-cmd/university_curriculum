Макетирование документа

По мере развития броузеров широкие возможности таблиц стилей реализуются все более полно. За последние несколько лет броузеры Mozilla и Opera значительно усилили поддержку CSS, однако Internet Explorer в этом отношении несколько отстает. Тем не менее можно сказать, что все новейшие версии популярных Web-броузеров поддерживают таблицы стилей достаточно хорошо.

Поэтому можно смело переходить на использование таблиц стилей не только для задания стилей текста, но и для представления документов методами макетирования на основе CSS. В документе "Блочная модель форматирования документа" рассмотрен общий формат отображения блоков документа.

Создание раскладки из трех столбцов

Ниже показан процесс создания документа, состоящего из трех столбцов, с использованием популярного метода, позиционирующего столбцы с помощью свойства position.

Создайте с помощью инструмента TopStyle новую таблицу стилей и сохраните ее в домашнем каталоге под именем maket.css.

Создайте для левого столбца селектор идентификатора с именем navAlpha. Создайте правило, позиционирующее левый столбец абсолютно, с помощью свойства position и значения absolute. Добавьте параметры позиционирования в пикселях.

#navAlpha {
	position:absolute;
	top:20px;
	left:20px;
	border:1px dashed black;
	background-color:#eee;
	padding:10px;
	width:128px;
	}

Вы создали левую полосу шириной 128 пикселей, которая будет позиционирована броузером таким образом, что слева останется свободная область шириной 20 пикселей, а сверху - шириной 20 пикселей. При необходимости, можно изменить эти значения. Для наглядности для левой полосы созданы также рамка толщиной 1 пиксель и изменен цвет фона ее объемлющего прямоугольника.

Теперь создадим динамически изменяющийся центральный столбец. Для этого создадим селектор класса с именем content.

.content {
	position:relative; /* Объявлено положение "relative" для правильной работы (z-index). */
	width:auto;
	min-width:120px;
	margin:0px 210px 20px 170px;
	border:1px solid black;
	background-color:white;
	padding:10px;
	z-index:3; /* Это позволяет блоку content перекрывать правое меню в узких окнах хороших броузеров. */
	}

Вы создали центральный столбец с минимальной шириной 120 пикселей, который с помошью свойства width:auto; позиционируется посередине между левым и правым столбцами. Центральный столбец имеет рамку и белый цвет фона своего объемлющего прямоугольника. Кроме того для него определены полосы - сверху размером 0 пикселей, справа размером 210 пикселей, снизу размером 20 пикселей и слева размером 170 пикселей.

Создайте теперь для правого столбца селектор идентификатора с именем navBeta. Создайте правило, позиционирующее правый столбец абсолютно, с помощью свойства position и значения absolute. Добавьте параметры позиционирования в пикселях

#navBeta {
	position:absolute;
	top:20px;
	right:20px;
	border:1px dashed black;
	background-color:#eee;
	padding:10px;
	z-index:1;
	width:168px;
	}

Для реализации созданной раскладки применим дескрипторы div с соответствующими идентификаторами. В таблицу стилей maket.css, кроме того, включены описания для некоторых стандартных элементов документа. Вид созданного документа можно увидеть здесь. Раскладка из двух столбцов - меню слева

Ниже показан процесс создания документа, состоящего из двух столбцов (меню слева), с использованием популярного метода, позиционирующего столбцы с помощью свойства position.

Создайте с помощью инструмента TopStyle новую таблицу стилей и сохраните ее в домашнем каталоге под именем maketl.css.

Создайте для левого столбца селектор идентификатора с именем Menu. Создайте правило, позиционирующее левый столбец абсолютно, с помощью свойства position и значения absolute. Добавьте параметры позиционирования в пикселях.

#Menu {
	position:absolute;
	top:100px;
	left:20px;
	padding:10px;
	background-color:#eee;
	border:1px dashed #999;
	line-height:17px;
	width:150px;
	}

Допускаются отрицательные размеры границ, но обозреватели могут накладывать свои ограничения.

Вы создали левую полосу шириной 150 пикселей, которая будет позиционирована броузером таким образом, что слева останется свободная область шириной 20 пикселей, а сверху - шириной 100 пикселей. При необходимости, можно изменить эти значения. Для наглядности для левой полосы созданы также рамка толщиной 1 пиксель и изменен цвет фона ее объемлющего прямоугольника.

Теперь создадим динамически изменяющийся столбец содержимого (контент). Для этого создадим селектор идентификатора с именем content.

#Content {
	margin:0px 50px 50px 200px;
	padding:10px;
	}

Вы создали столбец содержимого, который не имеет рамки. Кроме того для него определены полосы - сверху размером 0 пикселей, справа размером 50 пикселей, снизу размером 50 пикселей и слева размером 200 пикселей, а также отступы со всех сторон размером 10 пикселей.

Создайте теперь для заголовка документа селектор идентификатора с именем Header. Добавьте параметры позиционирования в пикселях

#Header {
	margin:50px 0px 10px 0px;
	padding:17px 0px 0px 20px;
	border-style:solid;
	border-color:black;
	border-width:1px 0px; /* верхняя и нижняя рамки: 1px; левая и правая рамки: 0px */
	line-height:11px;
	background-color:#eee;
	height:14px; /* правильное значение высоты */
	}

Вы создали заголовок документа, который имеет высоту 14 пикселей и имеет только верхнюю и нижнюю рамки толщиной 1 пиксель и изменен цвет фона ее объемлющего прямоугольника. Кроме того для него определены полосы - сверху размером 50 пикселей, справа размером 0 пикселей, снизу размером 10 пикселей и слева размером 0 пикселей, а также отступы - сверху размером 17 пикселей, справа размером 0 пикселей, снизу размером 0 пикселей и слева размером 20 пикселей.

Для реализации созданной раскладки применим дескрипторы div с соответствующими идентификаторами. В таблицу стилей maketl.css, кроме того, включены описания для некоторых стандартных элементов документа. Вид созданного документа можно увидеть здесь. Листинг XHTML документа, использующего созданную раскладку, можно просмотреть здесь.

Раскладка с использованием свойства float

Рассматриваемая методика приобретает все большую популярность в качестве простого способа макетирования на основе CSS.

Определим левый и правый столбцы с помощью селекторов id и свойства float. Для этого создайте с помощью инструмента TopStyle новую таблицу стилей и сохраните ее в домашнем каталоге под именем maketf.css.

Добавьте в него селектор #content для левого столбца, в котором будет расположен основной текст документа. Создайте правило на основе свойства float со значением left. Теперь установите ширину столбца в процентах и определите правую полосу, обеспечивающую свободное пространство между текстом левого столбца и правым столбцом.

#content {
   float: left;
   width: 70%;
   margin-right: 25px;
}

Затем создайте селектор #right и правило для правого столбца. В данном примере определено только правило, устанавливающее свободное пространство (отступ) сверху, однако вы можете добавить в него любые другие правила.

#right {
   padding-top: 20px;
}

Для реализации созданной раскладки применим дескрипторы div с соответствующими идентификаторами. .
